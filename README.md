# AI Resume Analyzer 🚀

A web application that analyzes resumes and provides a score based on detected skills.

## 🔧 Tech Stack
- Python (Flask)
- HTML, CSS
- PyPDF2

## ✨ Features
- Upload resume (PDF)
- Extract skills automatically
- Score generation based on skills

## 📸 Output
- Resume Score (out of 100)
- Skills detected

## 🚀 How to Run
1. Clone repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open: http://127.0.0.1:5000
